# 🏥 Pharmacy Sales Analysis

## 📌 Overview
This project analyzes pharmacy sales data to identify trends in drug sales, customer spending, and inventory risks.

## 🛠 Tools Used
- MySQL

## 📊 Dataset
Simulated pharmacy sales dataset including:
- Drug name
- Quantity sold
- Price
- Sale date
- Expiry date

## 🔍 Key Analysis
- Top-selling drugs
- Monthly revenue trends
- Customer spending patterns
- Expired drug detection

## 💡 Key Insights
- Paracetamol is the most sold drug
- Revenue increases over time
- A few customers contribute most of the revenue
- Some drugs are expired and require attention

## 🎯 Conclusion
This analysis helps improve decision-making in inventory management and sales strategy in a pharmacy setting.
